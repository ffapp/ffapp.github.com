富士康将发布搭载Firefox OS的设备

更新日期：2013-06-20|Tag：FirefoxOS(2)富士康(1)火狐(1)手机(1)
  • 软件介绍

 

 

　　Firefox OS在前进。

　　据 Engadget 报道，富士康正式同 Mozilla 达成合作协议，将于下周（六月 3 号）发布至少一款 Firefox OS 设备，至于这款设备到底是以富士康为品牌名或者只是富士康为其他合作伙伴代工生产就不得而知了。

　　富士康作为现今最大的代工厂，为多家 IT 公司提供代工生产服务，苹果可能是这些主顾中订单给的最多的一家。只是好时不再，苹果渐渐有了熟透的感觉，富士康也开始考虑是不是得再找颗大树了，比如 Mozilla。

　　Mozilla 的 Firefox OS 在第三世界的新兴市场有一定机会，上月开售的第一款手机反响也不错。富士康如果能够加盟，将为这个联盟带来强大的供应链和生产制造技术，在低价手机市场为 Firefox OS 争来更多胜算。

　　除了寻找新的合作伙伴之外，富士康还有其他扩张的动作，比如制造销售自有品牌的各种配件（耳机、键盘和缆线等），据华尔街报道，富士康还有意在云计算和连锁零售店上寻找出路。

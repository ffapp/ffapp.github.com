Android还不错 但Firefox OS将会改变手机未来

更新日期：2013-09-01|Tag：
  • 软件介绍
Firefox 近日宣布 Firefox OS 的智能手机正式上市！(截至本篇文章发表为止，已有西班牙与波兰的电信营运商开始贩售。)
 
    如果你所在地的市场还看不到 Firefox OS 手机，你看好 Firefox OS 会成功吗？我们让一位给予正面答案的专家来为大家解释。
 

 
    欢迎海尔曼 (Christian Heilmann)！
 
    Q: 先谈谈你和 Firefox OS 的渊源？
 
    A: 我本来就参与 Firefox OS 的相关工作，我协助开发者开发他们的 Apps，也花很多时间写了相关文章、设计 Demo、与新闻媒体相谈，甚至只要某个人在我旁边站上几分钟 (不然就是搭飞机时坐我隔壁)，我都会跟对方聊上几句 Firefox OS。
 
    Q: 为什么你觉得 Firefox OS 很重要？
 
 
    A: 有很多原因。
 
    Firefox OS 彻底终结了“移动上网只限西方世界的有钱人才用得起”的刻板概念。没错，住在美国或英国的人或许每半年就会换一支闪亮的新手机，但这不是所有人都付担得起的。即使可以，在许多国家，没有信用卡也不能购买任何手机上的Apps。这现象不仅存在精英主义式的不公平，也与互联网的理念背道而驰。Firefox OS 手机符合多数人的消费能力，更能透过预付费用或电信帐单代缴的方式购买 Apps。
 
    Firefox OS 并不要求一个十分快速、稳定、随时处于连网状态的网络环境。每次出国旅行时，我都会开始讨厌自己的 Android 手机 (当然它还是有其它讨人喜欢的特色)。因为我们不可能花费昂贵的漫游费，也不太想用速度缓慢又信号微弱的 WiFi 连线，随时下载大型的应用更新。而 Firefox OS 没有任何原生 Apps，且包含系统 Apps 在内的所有应用App，都是以 CSS、HTML、JavaScript 撰写而成。因此应用容量不仅比较小，更新也不大。只会更新部分的 App，而不会以整个 App 为单位进行更新。
 
    Firefox OS 就是你的随身 Web。Firefox OS 其实就是 Firefox，顶多再加上Linux 核心。当我开启可完整支持手机浏览的网站时，不会再看到系统要求我下载原生 App。
 
    Firefox OS 就是 HTML5 平台。对开发者来说，我们的 HTML5 解决方案终于是这个环境中的高级公民 (The first class citizens) 了！Firefox OS 中的 HTML5 Apps 可以完整发挥硬件的效能，而不再局限于执行缓慢的网页。
 
    Firefox OS 完全通过互联网发布 Apps。在 Marketplace 中，使用者能通过手动搜寻或 App 评分等级，找到我们所制作的 Apps。同样方式可用于搜寻特定的歌曲、影片、足球队伍，甚至一道菜的食谱。发掘 App 的过程充满乐趣，而且就像使用互联网或寻找网站一样简单。
 
    Firefox OS 还有太多令人开心的特色，让我兴奋得像跳进糖果堆的 5 岁小孩一样！

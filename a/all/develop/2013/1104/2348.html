火狐小众能出彩Firefox OS桌面模拟器

更新日期：2013-11-04|Tag：
  • 软件介绍
正当苹果iOS和谷歌Android战况正烈，微软WP在旁围观的时候，2013年的手机智能系统圈又已经诞生了两位新成员：Ubuntu Touch和Firefox OS。不过这两款新生的智能手机系统并不大众，很多人手机用户连它们的身影都没有见过。其中主打HTML5应用的Firefox OS系统被Mozilla官方大力推广，为此Mozilla也推出了此款手机系统的桌面模拟器，让更多用户能够在桌面端揭开Firefox OS的面纱。
 
Firefox OS手机操作系统
　　作为Firefox OS的桌面模拟器，B2G Desktop Client（另称为Firefox OS Desktop Client）是一款应用于Windows桌面的Firefox OS模拟器。它被Mozilla直接编译成为可执行的exe文件程序，此应用程序无需安装，解压后直接使用。Firefox OS系统是驴是马，溜出来看看就知道！下面我们就通过Firefox OS Desktop Client在桌面端来体验一下Firefox OS。
　　点击下载：Firefox OS Desktop Client
　　前文已经说到Firefox OS Desktop Client是免安装软件程序，下载完成后大家解压后打开b2g.exe即可使用。首次使用，Firefox OS Desktop Client会有一个类似新手机激活的向导界面，如下图：
